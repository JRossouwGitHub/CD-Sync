# CD Sync
A passion project that aims to better coordinate cooldown timers across the team.

## Contents
- Stack
- Dependencies
- How To
- Credits

## Stack
This project uses:
- Python (frontend)
- JavaScript (backend)

## Dependencies
### 1. Python
Version 3.11.4 or higher [(*download latest*)](https://www.python.org/downloads/)
```
pip install websockets
# python -m pip install websockets #this may be needed in some situations
```

### 2. NodeJS
Version 16.16.0 or higher [(*download latest*)](https://nodejs.org/en/download)
```
npm install
```

## How To
- Run the backend websocket server
- Connect with the front end

### Run the backend websocket server
```
npm run start
# npm run dev #this just enabled hot reload
```

### Connect with the front end
```
py Tests.py
```
